🔗 המפה המשותפת ב-Google My Maps

🗺️ פתח את המפה
אחרי שמעלים את trip-map.kml ל-My Maps ולוחצים "שיתוף", מדביקים כאן את הקישור — הוא נשמר במכשיר הזה ויהפוך לכפתור. (רוצים שיהיה קבוע בקובץ לכולם? תנו לי את הקישור ואטמיע אותו.)
⬇️ trip-map.kml (כל השכבות) 🔵 lodging.csv 🔴 attractions.csv ⚪ transport.csv

הוראות ייבוא ושיתוף

  1. היכנסו ל-mymaps.google.com ולחצו "צור מפה חדשה".
  2. לחצו "ייבוא" תחת שם השכבה והעלו את trip-map.kml. אם השכבות לא נשמרות — צרו 3 שכבות וייבאו כל CSV בנפרד (Latitude/Longitude כמיקום, Name ככותרת).
  3. עצבו את צבע הפין לפי קטגוריה (כחול/אדום/אפור) ושנו את שם השכבה לפי הצורך.
  4. לחצו "שיתוף" → "כל מי שיש לו הקישור — צפייה" → העתיקו את הקישור ושלחו למשפחה. ניתן לפתוח ב-Google Maps בנייד ולהדפיס.

פיני הלינה ממוקמים לפי אזור/עיר ("מיקום מקורב"); לאחר בחירת נכס סופי כדאי לעדכן את הפין למיקום המדויק. כל המיקומים מקודדים ב-UTF-8 לתמיכה מלאה בעברית.
