הוראות ייבוא ושיתוף

  1. היכנסו ל-mymaps.google.com ולחצו "צור מפה חדשה".
  2. לחצו "ייבוא" תחת שם השכבה והעלו את trip-map.kml. אם השכבות לא נשמרות — צרו 3 שכבות וייבאו כל CSV בנפרד (Latitude/Longitude כמיקום, Name ככותרת).
  3. עצבו את צבע הפין לפי קטגוריה (כחול/אדום/אפור) ושנו את שם השכבה לפי הצורך.
  4. לחצו "שיתוף" → "כל מי שיש לו הקישור — צפייה" → העתיקו את הקישור ושלחו למשפחה. ניתן לפתוח ב-Google Maps בנייד ולהדפיס.
